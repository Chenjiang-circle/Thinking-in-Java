## lambda表达式

#### 简介:

Lambda表达式，也可称为闭包。它允许把函数作为一个方法的参数（函数作为参数传递进方法中）

#### lambda表达式实例：
```java
// 1. 不需要参数，返回值为5
() -> 5

// 2. 接受一个参数（数字类型），返回其二倍的值
x -> 2 * x

// 3. 接受两个参数，并返回他们的差值
(x, y) -> x - y

// 4. 接收2个int型整数，并返回他们的和
(int x, int y) -> x + y

// 5. 接受一个String对象，并在控制台打印，不返回任何值
(String s) -> System.out.print(s)
```
